# NumPy Store Sales Analyzer
A beginner_friendly NumPy project
that analyzes weeklt store sales
data.

## Features
- Calculate total weekly sales
- Calculate average daily sales
- Find highest sales day
- Find lowest sales day
- Identify days with sales above 15000
- Count days with sales above average
- Calculate diff b/w highest and lowest
- Apply bonus on high-sales days
- Generate final sales report

## Technologies Used
- Python
- NumPy

# Learning Outcomes
- NumPy Arrays
- Indexing
- Filtering
- Loops
- Conditional Statements
- Statistical Functions
- Data Analysis Basics
